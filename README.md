# Portal de Gestão de Empreendimentos - Santa Catarina (SCTEC)

## 📌 Descrição do Projeto
Este projeto é uma aplicação Web (Front-end) desenvolvida para o processo seletivo da trilha **IA para DEVs** do programa SCTEC. A solução consiste em um sistema CRUD para gerenciamento de empreendimentos no estado de Santa Catarina, com foco especial no setor logístico, integrando conhecimentos de regras de negócio reais à tecnologia web moderna.

## 🚀 Tecnologias Utilizadas
- **HTML5 & CSS3**: Estrutura e estilização.
- **JavaScript (ES6+)**: Lógica de programação e manipulação da DOM.
- **Bootstrap 5**: Framework CSS para garantir responsividade e agilidade no design.
- **LocalStorage**: Mecanismo de persistência de dados no navegador.

## 🏗️ Estrutura do Projeto
O projeto segue uma organização modular para garantir a legibilidade e manutenção do código:
- `/css`: Arquivos de estilização.
- `/js`: Lógica dividida entre persistência (`storage.js`) e interface (`main.js`).
- `index.html`: Ponto de entrada da aplicação.

## 🔧 Como Executar
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Não há necessidade de servidores externos ou bancos de dados configurados, pois utilizamos o LocalStorage.
